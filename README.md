## How to Install ?
On your Linux Bash:

```md 
wget https://github.com/namespaceahr/MongoDB_Ubuntu/blob/master/mongo.sh
```

Give execution permission

```md
chmod +x mongo.sh ./install.sh
```

Now Run Install 

```md
./mongo.sh
```
